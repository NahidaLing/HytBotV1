## 已知问题

1. 该版本是通过WebSocket和MCC通信 其中WebSocket容易莫名其妙断掉

2. 该版本在执行进入游戏指令后是直接进入游戏 没有进行任何检查

3. 该版本主程序极小概率莫名其妙退出

# 使用方法

1. 自己安装 易语言~5.93破解版~ 构建

2. 在cmd中打开mcc 如果正常打开没事 如果提示需要依赖请下载

3. 把主程序放在MCC上层目录

4. 运行
